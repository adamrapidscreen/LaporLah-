# E1-S1: Initialize Next.js 15 Project with TypeScript, Tailwind CSS 4, and shadcn/ui

## Description
Bootstrap the LaporLah project with Next.js 15 (App Router), TypeScript strict mode, Tailwind CSS 4, and shadcn/ui component library. Configure ESLint, Prettier, path aliases, and install all required shadcn/ui primitives.

## Dependencies
- Stories: None (first story)
- Files: None (greenfield project)

## Tasks
1. `package.json`: Run `npx create-next-app@latest` with TypeScript, Tailwind CSS, App Router, `src/` directory, and `@/` import alias
2. `tsconfig.json`: Verify/set `"strict": true`, path alias `"@/*": ["./src/*"]`, `"target": "ES2022"`
3. `next.config.ts`: Create base Next.js config (TypeScript config file)
4. `.eslintrc.json`: Extend `next/core-web-vitals` and `next/typescript`, add rules for named exports preference and import ordering
5. `.prettierrc`: Configure `semi: true`, `singleQuote: true`, `trailingComma: "es5"`, `tabWidth: 2`, `printWidth: 100`
6. Install Tailwind CSS 4 and configure with `@import "tailwindcss"` directive
7. Install shadcn/ui via `npx shadcn@latest init` with New York style, Emerald theme
8. Install all required shadcn/ui components: `npx shadcn@latest add button card badge input textarea select dialog dropdown-menu avatar tabs progress toast separator sheet skeleton popover tooltip toggle`
9. Install additional project dependencies: `leaflet react-leaflet browser-image-compression @supabase/supabase-js @supabase/ssr next-themes lucide-react zod`

## Implementation Notes
```bash
# Project initialization
npx create-next-app@latest laporlah --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"

# shadcn/ui init
npx shadcn@latest init

# Bulk install shadcn components
npx shadcn@latest add button card badge input textarea select dialog dropdown-menu avatar tabs progress toast separator sheet skeleton popover tooltip toggle

# Additional dependencies
npm install @supabase/supabase-js @supabase/ssr next-themes lucide-react zod
npm install leaflet react-leaflet browser-image-compression
npm install -D @types/leaflet
```

### Path Alias Configuration (tsconfig.json)
```json
{
  "compilerOptions": {
    "strict": true,
    "target": "ES2022",
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

### Import Order Convention
```
1. React / Next.js
2. Third-party libraries
3. @/components/ui (shadcn primitives)
4. @/components (app components)
5. @/lib (utilities, actions, hooks)
6. Types
```
