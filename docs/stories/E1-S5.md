# E1-S5: Set Up Root Layout with Fonts, Theme Provider, and Global CSS Tokens

## Description
Create the root application layout with Inter and JetBrains Mono fonts loaded via `next/font/google`, dark/light theme support via `next-themes`, comprehensive CSS custom properties for the design system, and the `cn()` utility for conditional class merging.

## Dependencies
- Stories: E1-S1, E1-S4
- Files: `package.json`, `src/app/manifest.ts`

## Tasks
1. `src/app/layout.tsx`: Create root layout with HTML lang="en", `next/font/google` for Inter (sans) and JetBrains Mono (mono), `<ThemeProvider>` from `next-themes` wrapping children, metadata (title, description, manifest link)
2. `src/styles/globals.css`: Create global stylesheet with Tailwind v4 `@import "tailwindcss"` directive, all CSS custom properties for dark and light themes (colors, radii, spacing), base layer resets
3. `src/lib/utils/cn.ts`: Create `cn()` utility combining `clsx` and `tailwind-merge` for conditional class names

## Implementation Notes

### Root Layout (`src/app/layout.tsx`)
```typescript
import type { Metadata } from 'next';
import { Inter, JetBrains_Mono } from 'next/font/google';
import { ThemeProvider } from 'next-themes';

import '@/styles/globals.css';

const inter = Inter({
  subsets: ['latin'],
  variable: '--font-inter',
  display: 'swap',
});

const jetbrainsMono = JetBrains_Mono({
  subsets: ['latin'],
  variable: '--font-jetbrains-mono',
  display: 'swap',
});

export const metadata: Metadata = {
  title: 'LaporLah! â€” Community Issue Reporting',
  description: 'Community-driven civic issue reporting for Malaysia',
  manifest: '/manifest.json',
};

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body className={`${inter.variable} ${jetbrainsMono.variable} font-sans antialiased`}>
        <ThemeProvider attribute="class" defaultTheme="dark" enableSystem disableTransitionOnChange>
          {children}
        </ThemeProvider>
      </body>
    </html>
  );
}
```

### cn() Utility (`src/lib/utils/cn.ts`)
```typescript
import { type ClassValue, clsx } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

### Font Usage Convention
- Body text: `font-sans` (Inter via CSS variable `--font-inter`)
- Stats/numbers: `font-mono` (JetBrains Mono via CSS variable `--font-jetbrains-mono`)
