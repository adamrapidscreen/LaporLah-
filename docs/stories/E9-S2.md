# E9-S2: Build Admin Stats Cards

## Description
Create the admin dashboard overview page as a Server Component that displays stat cards showing total reports, reports breakdown by status, and flagged items count. Numbers use `font-mono` for a dashboard aesthetic using shadcn Card components.

## Dependencies
- Stories: E9-S1 (admin layout with role guard)
- Files: `src/app/admin/layout.tsx`, `src/components/ui/card.tsx`

## Tasks
1. `src/app/admin/page.tsx`: Create Server Component that:
   - Calls `getAdminStats()` from `@/lib/actions/admin` (from E9-S5)
   - Renders stat cards in a responsive grid (2 cols on mobile, 4 cols on desktop)
   - Stat cards: Total Reports, Open, In Progress, Resolved, Closed, Flagged Items
   - Each card shows a label and a large `font-mono` number
   - Use shadcn `Card`, `CardHeader`, `CardTitle`, `CardContent`
2. `src/components/admin/stat-card.tsx`: Create reusable stat card component accepting `title`, `value`, `description?`, and optional `icon` props

## Implementation Notes

### Dashboard Page
```tsx
// src/app/admin/page.tsx
import { getAdminStats } from '@/lib/actions/admin';
import { Card, CardContent, CardHeader, CardTitle } from '@/components/ui/card';
import { FileText, AlertTriangle, Clock, CheckCircle, XCircle, Flag } from 'lucide-react';

export default async function AdminDashboardPage() {
  const stats = await getAdminStats();

  return (
    <div className="space-y-6">
      <h2 className="text-lg font-semibold">Overview</h2>
      <div className="grid grid-cols-2 gap-4 md:grid-cols-3 lg:grid-cols-6">
        <StatCard title="Total Reports" value={stats.totalReports} icon={FileText} />
        <StatCard title="Open" value={stats.open} icon={AlertTriangle} />
        <StatCard title="Acknowledged" value={stats.acknowledged} icon={Clock} />
        <StatCard title="In Progress" value={stats.inProgress} icon={Clock} />
        <StatCard title="Resolved" value={stats.resolved} icon={CheckCircle} />
        <StatCard title="Flagged" value={stats.flaggedCount} icon={Flag} />
      </div>
    </div>
  );
}
```

### Stat Card Component
```tsx
// src/components/admin/stat-card.tsx
import { Card, CardContent, CardHeader, CardTitle } from '@/components/ui/card';
import { type LucideIcon } from 'lucide-react';

interface StatCardProps {
  title: string;
  value: number;
  icon?: LucideIcon;
  description?: string;
}

export function StatCard({ title, value, icon: Icon, description }: StatCardProps) {
  return (
    <Card>
      <CardHeader className="flex flex-row items-center justify-between space-y-0 pb-2">
        <CardTitle className="text-sm font-medium text-muted-foreground">{title}</CardTitle>
        {Icon && <Icon className="h-4 w-4 text-muted-foreground" />}
      </CardHeader>
      <CardContent>
        <div className="text-2xl font-bold font-mono">{value}</div>
        {description && <p className="text-xs text-muted-foreground mt-1">{description}</p>}
      </CardContent>
    </Card>
  );
}
```

### AdminStats Type
```typescript
interface AdminStats {
  totalReports: number;
  open: number;
  acknowledged: number;
  inProgress: number;
  resolved: number;
  closed: number;
  flaggedCount: number;
}
```

### Styling Notes
- Grid: `grid-cols-2 md:grid-cols-3 lg:grid-cols-6` for responsive layout
- Numbers: `text-2xl font-bold font-mono` for dashboard aesthetic
- Labels: `text-sm font-medium text-muted-foreground`
- Icons from `lucide-react` at `h-4 w-4`
