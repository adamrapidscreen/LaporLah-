## Getting Started

### Prerequisites
- Node.js 18+
- npm or pnpm
- Supabase account (free tier)

### Installation
1. Clone the repository
2. Install dependencies: `npm install`
3. Copy environment file: `cp .env.local.example .env.local`
4. Fill in Supabase credentials (see Environment Variables below)
5. Set up Supabase: enable Google OAuth, create Storage bucket "report-photos"
6. Run database migrations in Supabase SQL editor
7. Seed the database: `npm run seed`
8. Start development server: `npm run dev`
