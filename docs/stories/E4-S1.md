# E4-S1: Build Report Detail Page

## Description
Create the report detail page as a Server Component that displays the full report information including photo, status stepper, title, category, map location, description, creator info, and follow button. Include a loading skeleton for the page.

## Dependencies
- Stories: E3-S4 (reports server actions), E3-S5 (report card components)
- Files: `src/lib/actions/reports.ts`, `src/components/reports/report-card.tsx`

## Tasks
1. `src/app/report/[id]/page.tsx`: Create Server Component that fetches report by ID with user join from Supabase, displays:
   - Full photo at 16:9 aspect ratio
   - Status stepper component (from E4-S2)
   - Title and category tag (Badge)
   - Leaflet map with pin via dynamic import (`ssr: false`) using `location-display.tsx`
   - Description body
   - Creator info section (avatar + name + formatted date)
   - Follow button placeholder
   - Handle 404 with `notFound()`
2. `src/components/map/location-display.tsx`: Create Client Component that renders a static Leaflet map with a marker at the report's coordinates. Use `dynamic(() => import(...), { ssr: false })` pattern. Accept `lat`, `lng`, and optional `zoom` props.
3. `src/app/report/[id]/loading.tsx`: Create loading skeleton with shimmer animation matching the detail page layout (photo skeleton, status dots, text blocks, map placeholder).

## Implementation Notes

### Page Component Structure
```tsx
// src/app/report/[id]/page.tsx
import { createServerClient } from '@/lib/supabase/server';
import { notFound } from 'next/navigation';
import { Badge } from '@/components/ui/badge';
import { Card } from '@/components/ui/card';
import { StatusStepper } from '@/components/reports/status-stepper';
import { FollowButton } from '@/components/shared/follow-button';
import dynamic from 'next/dynamic';

const LocationDisplay = dynamic(
  () => import('@/components/map/location-display').then(mod => mod.LocationDisplay),
  { ssr: false }
);

interface ReportDetailPageProps {
  params: Promise<{ id: string }>;
}

export default async function ReportDetailPage({ params }: ReportDetailPageProps) {
  const { id } = await params;
  const supabase = await createServerClient();
  const { data: report } = await supabase
    .from('reports')
    .select('*, user:users(*)')
    .eq('id', id)
    .single();

  if (!report) notFound();

  // render full detail layout
}
```

### Location Display Component
```tsx
// src/components/map/location-display.tsx
'use client';

import { MapContainer, TileLayer, Marker } from 'react-leaflet';
import 'leaflet/dist/leaflet.css';

interface LocationDisplayProps {
  lat: number;
  lng: number;
  zoom?: number;
}

export function LocationDisplay({ lat, lng, zoom = 15 }: LocationDisplayProps) {
  return (
    <MapContainer center={[lat, lng]} zoom={zoom} className="h-48 w-full rounded-lg">
      <TileLayer url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png" />
      <Marker position={[lat, lng]} />
    </MapContainer>
  );
}
```

### Photo Display
- Use `next/image` with `aspect-ratio: 16/9`, `object-cover`, full width
- Fallback placeholder if no photo URL
